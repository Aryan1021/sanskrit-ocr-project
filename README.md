# Sanskrit OCR using Deep Learning

This project implements an Optical Character Recognition (OCR) system for recognizing Sanskrit characters written in the Devanagari script using Deep Learning.

The system will use a Convolutional Neural Network (CNN) to recognize characters from images and evaluate the model using accuracy, precision, recall, and F1-score.
